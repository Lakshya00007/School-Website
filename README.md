# 🏫 The Delhi Public School Demo website 

A fully functional and responsive multi-page website for **The Delhi Public School Society (DPS)**, 
built with pure HTML, CSS, and vanilla JavaScript. Inspired by [dpsfamily.org](https://dpsfamily.org).

---

## 🌐 Live Preview

> Open `dps-society-website.html` directly in any browser — no build tools or dependencies required.

## 📄 Pages Included

| Page | Description |
|------|-------------|
| 🏠 Home | Hero slider, stats bar, legacy timeline, featured schools, vision, events |
| 🏛️ DPS Society | Vision & Mission, Chairman's & Vice Chairman's messages |
| 🏫 DPS Schools | Filterable school directory — Core / India / Abroad |
| 🏆 Hall of Fame | Achievement cards by school |
| 🖼️ Gallery | Photo grid with category filters and hover overlays |
| 📅 Events | Sports, Cultural & Language Festival listings |
| 📬 Contact | Contact form, office hours, address, and map link |

---

## ✨ Features

- 🎠 Auto-advancing hero slider with arrow and dot controls
- 📱 Fully responsive — mobile, tablet, and desktop
- ☰ Hamburger navigation menu for mobile
- 🔍 Filterable school directory (Core / India / Abroad)
- 📢 Animated announcement ticker bar
- 🎨 DPS brand colours — green, teal, navy, and gold
- 🔡 Playfair Display + DM Sans font pairing
- ⚡ Zero dependencies — pure HTML/CSS/JS, no frameworks
